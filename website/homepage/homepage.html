What is GitBuild?

GitBuild is a Git-like version control system for Minecraft builds, modeled after MCGit. It allows you to track changes to your builds, manage commits, and collaborate with other players.

Version Control

Track all changes to your builds with commit history

Remote Operations

Push, pull, and clone repositories with ease

Block Tracking

Automatic block change tracking and staging

Revert Capability

Revert your world to any previous commit
